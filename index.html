📖 About the project

This application provides a modified version of the standard Audiobookshelf Android client. It includes specific patches for Cloudflare Zero Trust support. Many users hosting their own audiobook servers face challenges when trying to connect securely from outside their home network. This version solves that problem by enabling Single Sign-On (SSO) and encrypted credential storage. It allows your device to talk to your server securely without opening complex ports on your router.

🚀 Key features

  • Integrated Cloudflare Zero Trust authentication.
  • Automatic detection and routing for local network access.
  • Encrypted storage for your login information.
  • Support for custom security headers.
  • Smooth integration with existing Audiobookshelf servers.

⬇️ How to install

Follow these steps to set up the application on your Android device. Since this is an unofficial patched version, you must enable installations from unknown sources in your Android security settings.

  1. Open the release page to find the latest download files.
  2. Look for the file ending in .apk.
  3. Select the file to start the download directly to your phone.
  4. If your phone asks for permission to install apps from your browser, select "Allow".
  5. Open the downloaded file and select Install.

⚙️ Getting connected

Once you install the app, you will see a login screen. Enter the URL of your Audiobookshelf server. If you use Cloudflare Zero Trust, the app will redirect you to your authentication provider. Log in with your credentials once. The app will save these securely so you do not need to repeat this step every time you open the app.

The app automatically detects if you are home on your Wi-Fi or away on mobile data. It chooses the best connection path to keep your stream stable and fast.

🛡️ Security details

We prioritize your privacy. The patches included here do not send your data to any third-party servers. All authentication happens strictly between your device, your identity provider, and your personal Audiobookshelf server. We store all sensitive tokens in the Android Keystore, which uses hardware-backed encryption to prevent unauthorized access to your account details.

🔧 Requirements

To use this application correctly, ensure your server meets the following criteria:

  • An active instance of the Audiobookshelf server software.
  • A Cloudflare Zero Trust account configured for your domain.
  • The Android device must run Android 8.0 or newer.
